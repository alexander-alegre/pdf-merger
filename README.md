# Setup

### 1:

```shell
mkdir end
```


### 2:

```shell
mkdir start
```

# Running

### 3: Put all your PDFs in the `start` folder

### 4: Activate virtual environment
```shell
python3 -m venv venv
source venv/bin/activate
```

### 5: Get dependencies
```shell
pip install PyPDF2
```


### 6: run
```shell
python main.py
```
